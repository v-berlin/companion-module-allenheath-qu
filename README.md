# companion-module-allenheath-qu

See [HELP.md](HELP.md) and [LICENSE](LICENSE) for more information about this module.

This module for the Allen & Heath QU can control parameters of the console
via MIDI commands over IP.

Created by Max Kiusso in 2021.  
Ported to Companion 4 (SDK v3 / `@companion-module/base`) in 2024.

## Compatibility

| Module version | Companion version |
|---------------|-------------------|
| 2.x           | Companion 3.x / 4.x (tested with 4.3.1) |
| 1.x           | Companion 2.x (legacy, no longer maintained) |

## Supported consoles

- Allen & Heath QU-16
- Allen & Heath QU-24
- Allen & Heath QU-32
- Allen & Heath QU-SB
- Allen & Heath QU-Pac

## Installation (Companion 4)

The module can be loaded as a **developer module** in Companion 4:

1. In Companion 4 Settings → Developer modules, add the path to this folder.
2. Or: copy this folder into the `develop` directory of your Companion data folder.
3. Restart Companion and add a new connection of type **Allen & Heath QU**.
